# Méthode épreuve Front – 4h (+ tiers temps)

## 1. Lecture & cadrage (20–30 min)
- Lire tout le sujet
- Lister les fonctionnalités demandées
- Identifier : HTML / CSS / JS / fetch
- Faire un plan rapide de la page

## 2. Structure HTML (45 min)
- Écrire le HTML complet
- Header / nav / main / footer
- Sections claires
- Formulaire en place
- Footer sticky (flex)

Objectif : page fonctionnelle sans CSS

## 3. CSS global & layout (1h)
- Reset + variables CSS
- Layout principal (flex / grid)
- Palette couleurs
- Footer en bas de page
- Responsive de base

Objectif : site propre, lisible, utilisable

## 4. JavaScript interactions (1h)
- Menu mobile
- Ajout d’éléments dynamiques
- Validation formulaire
- Suppression / modification

Objectif : interactions visibles et fiables

## 5. Fetch / AJAX (30–45 min)
- fetch API
- affichage dynamique
- gestion erreur
- loader ou message si vide

## 6. Finalisation (30 min)
- Commentaires
- Console propre
- Test responsive
- Vérification accessibilité
